Assistente de IA (Claude)

Envie o resumo do seu mês para a IA analisar gastos, riscos e sugerir melhorias. Esse recurso é compartilhado entre todas as contas do app: a chave da Anthropic fica guardada em segredo em uma function no Supabase, nunca no navegador.

ℹ️ Disponível para qualquer pessoa logada no app. A chave da API não fica no seu navegador nem é visível no código-fonte — quem chama a IA é a function ia-proxy rodando no seu projeto Supabase. Existe um limite diário de chamadas por conta para evitar uso excessivo da chave compartilhada.

Como publicar a function (uma vez só, feito por quem administra o app)

Isso é configuração de backend, não precisa repetir por usuário. Depois de feito, todo mundo que fizer login já consegue usar a IA.

1. Instale a CLI do Supabase e faça login: supabase login 2. Na pasta do projeto: supabase functions new ia-proxy 3. Copie o código do arquivo ia-proxy-index.ts (baixado junto com este app) para supabase/functions/ia-proxy/index.ts 4. Guarde a chave da Anthropic em segredo: supabase secrets set ANTHROPIC_API_KEY=sk-ant-sua-chave-aqui 5. Publique: supabase functions deploy ia-proxy 6. Rode o SQL atualizado (aba Configuração > SQL do banco) para criar as tabelas ia_analises e ia_uso
